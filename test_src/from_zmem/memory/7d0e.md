# The package is a Dart CLI and builder tool built on args, yaml, path, build, and build_runner, and it targets Flutter consumers using slang-generated translation layers.

## Context

The repo itself is a Dart package, but its generated outputs are consumed by Flutter apps that already use slang and TranslationProvider.

## Content

Current key packages:

- `args` for CLI parsing
- `yaml` for `pvtro.yaml` parsing
- `path` for cross-platform path handling and POSIX import normalization
- `build` and `build_runner` for builder-mode generation
- `test` and `lints` for validation

## Consequences / notes

Documentation should describe the tool as a Dart package with Flutter-facing outputs, not as a Flutter package itself.
