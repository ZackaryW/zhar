# Generate a root integration file for multi-package slang apps by discovering package translation layers and emitting wrapper plus locale-sync helpers.

## Context

PVTRO runs from the consuming project's root package and inspects the package graph to find slang-enabled packages that already expose generated translation entrypoints.

## Content

The primary deliverable is a generated integration file for the root app.

That file exports:

- `pvtroWrapper({required Widget child})`
- `pvtroSyncPackageLocales(String rawLocale)`

## Consequences / notes

The project should stay focused on discovery and code generation for existing slang layers, not on re-implementing slang's translation model.
